# 📄 석·박사 연구 논문 (2011~2015)

> [← 포트폴리오 홈](https://github.com/Taekyo-Lee/portfolio)

**학위과정(석사·박사)의 정보이론 연구 성과입니다.**

석·박사 과정에서 **대수학(Algebra), 특히 유한체(Galois field)에 기초**해 두 축을 연구했습니다: **① 의사난수 수열의 상관(correlation) 이론**과 **② 대수 부호(오류정정부호)**. 두 분야는 유한체 대수라는 **같은 수학적 뿌리**를 공유합니다: m-수열의 순환천이 집합이 곧 심플렉스 부호(해밍 부호의 쌍대)이듯, 수열 설계와 오류정정부호는 하나의 대수 언어로 쓰인 두 응용입니다.

그중 **연구 성과(제1저자 논문)는 수열 분야**에서 냈습니다: Zadoff-Chu·GCL/ZCZ 수열의 상관 특성과 MIMO 채널추정용 수열 설계. (대수 부호는 산학과제로 NAND 플래시 ECC 설계에 적용 → [research](https://github.com/Taekyo-Lee/portfolio/tree/main/research) 참고.)

아래는 그 제1저자 논문입니다 (**국제 저널 2편 · 국제 학회 5편**).
